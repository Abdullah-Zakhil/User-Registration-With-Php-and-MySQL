User-Registration-With-Php-and-MySQL

This is a complete project (website) of user registration, log-in and send/receive message using Phpmyadmin and MySQL database. The project works as follows: • Users can register themselves. • First of all run the "register.html" file in the web browser or wamp server. • Enter your credentials to register yourself. • Check your info in MySQL database there will be a new entry • Now you can login • Only registered users can login. • After login, users can send a text message to anybody who is registered with the website. • Users can view the received messages.
